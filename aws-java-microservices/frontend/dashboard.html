🏗️ Arquitetura do Sistema

Este sistema usa arquitetura de microserviços com dois serviços independentes que se comunicam via API REST (HTTP). Hospedado na AWS Elastic Beanstalk.

  • Auth Service (porta 8081) — Autenticação e gestão de usuários
  • Catalog Service (porta 8082) — Gestão de clientes e produtos
  • Nginx Gateway (porta 80) — Roteamento reverso de APIs e Arquivos Estáticos

Padrões aplicados: MVC, GRASP (Controller, Creator, Expert, Alta Coesão, Baixo Acoplamento).
